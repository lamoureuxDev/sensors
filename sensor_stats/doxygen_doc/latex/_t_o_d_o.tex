The access of data should be controled, to avoid undefined behavior. To do so, semaphores or mutex (sem\+\_\+t) can be added to the sensor structure, stored in the shared memory. Before accessing to sensor data, we should use a sem\+\_\+wait(),waiting for the semaphore to be released. Then, use a sem\+\_\+post() once it is not modified.
